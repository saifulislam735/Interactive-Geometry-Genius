1.What is DOM and what is the purpose of DOM?

answer:The Document Object Model (DOM) is a programming API for HTML and XML documents. It defines the logical structure of documents and the way a document is accessed and manipulated.

2.How will you select HTML elements using DOM? Name the DOM methods.

answer:There are 5 ways in which you can select elements in a DOM using selectors. getElementsByTagName() getElementsByClassName() getElementById() querySelector() querySelectorAll()

3.What is event bubble?

answer:Event bubbling is a method of event propagation in the HTML DOM API when an event is in an element inside another element, and both elements have registered a handle to that event.

4.What is a callback function and why will you use it?

answer:A JavaScript callback is a function which is to be executed after another function has finished execution.
